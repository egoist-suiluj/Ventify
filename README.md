VENTIFY – Emotional Venting Application
---------------------------------------

Ventify is an idea created by Boss (Patricia Villas). Ako ay isa sa mga developers na napili niya
para mag-develop ng system at functionalities, habang si Boss Patricia naman ang author at
main designer ng konsepto at direction ng app. Siya ang naglatag ng purpose, flow, at core
experience ng Ventify.

BACKGROUND
-----------
Si Boss ay nakaranas ng matinding anxiety at minsang nagpakonsulta sa isang espesyalista.
Doon niya napansin na napakahaba ng proseso at matagal bago ma-accommodate ang isang
taong nangangailangan ng emotional release o professional support.

Dahil dito, naisip niya: bakit hindi gumawa ng isang application na pwedeng puntahan ng mga
taong gustong maglabas ng sama ng loob, galit, pagod, frustration, o kahit anong mabigat na
nararamdaman—nang walang nasasaktan at walang hinahandang explanation?

THE IDEA
---------
Ventify is designed with eight (8) emotional rooms. Bawat kwarto ay may sariling theme o
Purpose kung saan pwedeng pumasok ang user at i-vent out lahat ng laman ng loob nila.

Ito man ay:
- galit  
- sama ng loob  
- heartbreak  
- frustration  
- anxiety  
- disappointment  
- confusion  
- o kahit ano pang mabigat na emosyon  

Layunin ng Ventify na maging “safe punching bag” ng users: isang lugar kung saan nila
mailalabas ang bigat sa dibdib nang walang kaharap, walang judgment, at walang pressure.

PURPOSE OF THIS APP
--------------------
Ventify exists to provide a fast, accessible, and non-judgmental space for emotional release.
Para sa mga user na hindi agad ma-accommodate ng professionals, o para sa mga taong
nahihiyang maglabas ng nararamdaman sa ibang tao, Ventify serves as a relief channel.

Its mission is simple:
"Kung hindi mo masabi sa iba, dito mo pwedeng ilabas."
